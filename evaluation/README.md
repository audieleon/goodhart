# Evaluation Encodings

These encodings were **not** used during development of goodhart's rules, examples, or presets. They exist solely for independent evaluation of the tool's accuracy.

## Structure

- `eureka/` — 10 reward functions generated by GPT-4 via Eureka (Ma et al. 2024). Source code from [eureka-research.github.io](https://eureka-research.github.io/assets/reward_functions/). 5/10 rejected with CRITICAL findings.
- `heldout/` — 5 reward configurations from recent papers (NeurIPS 2024, IROS 2024, EWRL 2025) not referenced during tool development. 5/5 correct.

## Running

Each file is a standalone script:

```bash
python evaluation/eureka/eureka_ant.py
python evaluation/heldout/heldout_omnigrasp.py
```

## Relationship to the paper

These correspond to Table 3 (held-out evaluation) and Table 5 (Eureka screening) in the paper. The tool's built-in 57 examples live in `goodhart/examples/` and are entirely separate.
